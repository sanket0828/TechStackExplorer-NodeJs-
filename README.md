# Node.js Product Catalog (Learning Project)

A vanilla **Node.js** project built to understand how backend systems work **without using any frameworks**.

This application serves a small product catalog (tech tools instead of food items) using **server-side rendering**, HTML templates, and JSON data. It dynamically generates overview and product detail pages using only Node.js core modules.

- Dynamic HTML generation
- Conditional rendering based on data (e.g. Open Source badge)
- Defensive error handling to avoid runtime crashes
